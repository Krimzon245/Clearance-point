# Clearance Point 🏷️

A modern web platform for a local liquidation store that allows customers to browse inventory, place items on hold, and schedule in-person viewings and payments.

## 🚀 Current Project Scope (Phase 1: Local Only)
The initial release focuses entirely on local customers to drive foot traffic and simplify inventory turnover without shipping complexities.

### Core Features:
* **Inventory Browsing:** Customers can view available liquidation products, descriptions, and discounted pricing.
* **Reserve & Hold:** Users can "reserve" an item to set it aside so no one else can purchase it online.
* **Appointment Booking:** Upon placing a hold, the user selects a specific date and time slot to visit the physical store.
* **In-Person Inspection & Pay:** Customers inspect the item in person to ensure satisfaction before completing their purchase at the counter.

---

## 🗺️ Future Roadmap (Phase 2)
* **Canada-Wide Expansion:** Integrate shipping carrier APIs to allow orders to be shipped across Canada.
* **Online Payments:** Implement Stripe or PayPal to accept online credit/debit cards during the shipping phase.
